# EC²R — Site Web

Site vitrine de la méthode EC²R.

## Stack

- HTML5
- CSS3
- JavaScript vanilla

## Développement

Ouvrir `index.html` dans un navigateur, ou utiliser un serveur local :

```bash
python3 -m http.server 8080
```
